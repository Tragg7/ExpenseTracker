## Используемые технологии
- Java
- Spring boot
- Spring MVC
- Spring Security
- Spring Data (JPA)
- MySQL
- Thymeleaf
- Bootstrap

1. **Конфигурация БД:**
Установите Postgres и обновите application.properties файл с вашей конфигурацие БД(host, пароли).

3. **Сборка и запуск:**
`mvn clean install`.

4. **Запуска jar файла:**
`java -jar target/expenses-tracker.jar`.

5. **На странице:**
Откройте IP и перейдите на страницу `http://localhost:8080`.
